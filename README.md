# 🪓 Hangman Game (Python)

A console-based Hangman game where the player guesses a hidden word one letter at a time.

## 📌 Overview

Built for Day 7 of the *100 Days of Code: Python Bootcamp* by Dr. Angela Yu. The game tracks correct and incorrect guesses and displays ASCII art for each stage.

## 🛠️ Tech Used

- Python 3
- Loops, conditionals, and string/list handling

## 📚 What I Learned

- Basic game loop logic
- Managing game state using lists
- ASCII art and string updates

## 🚀 How to Run

```bash
python main.py
